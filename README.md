🛒 E-Commerce Sales Analysis

An end-to-end data analysis project exploring transactional e-commerce data to uncover revenue trends, customer behavior patterns, and product performance insights that drive strategic business decisions.

📌 Project Overview

This project analyzes historical e-commerce transaction data to answer key business questions such as:

What are the overall sales trends over time?

Which product categories generate the most revenue?

Who are the most valuable customers?

Which regions drive the highest sales?

What actionable strategies can improve revenue and retention?

The goal was to transform raw transactional data into meaningful, business-oriented insights using structured exploratory data analysis (EDA).

🎯 Business Objectives

Identify top-performing products and categories

Analyze revenue distribution across regions

Evaluate customer purchasing behavior

Detect seasonal sales patterns

Provide data-backed strategic recommendations

🧹 Data Cleaning & Preparation

Key preprocessing steps included:

Removing duplicate records

Handling missing values

Filtering cancelled transactions

Removing non-positive prices and quantities

Converting date columns into proper datetime format

Creating new derived features (e.g., total revenue)

These steps ensured data accuracy and reliability before analysis.
